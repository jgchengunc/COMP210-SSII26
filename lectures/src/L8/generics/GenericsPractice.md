# Generics Practice

## Instructions
1. In this package (`L8.generics`), create a new Pair class and copy the starter code.
Because of its generic types, a Pair can hold a pair of any two object types!
2. Write a main method inside the class. Use this on the following steps to create Pair objects,
test different parts of your code, print things out, and experiment!
3. Write a few more methods in Pair that you think would be useful or just interesting to you
   - Examples:
     - Getters
     - Setters
     - Data (field) manipulation
     - More complex value derivation from fields
     - Making two Pair objects interact
4. Write an interface that Pair implements
5. Experiment with keeping the class generic as well as setting specific object types when
implementing the interface
